My First Year Python Project
This repository contains my Python assignments and experiments.

How to run(Windows):
1. Create the environment: `python -m venv .venv`
2. Activate the environment: `.\.venv\Scripts\activate.bat`
3. Install requirements: `pip install -r requirements.txt`
4. Run the desired programs: python "Desired Program Name"

How to run(MacOS):
1. Create the environment: `python3 -m venv .venv`
2. Activate the environment: `source .venv/bin/activate`
3. Install requirements: `pip install -r requirements.txt`
4. Run the desired programs: python3 "Desired Program Name"
