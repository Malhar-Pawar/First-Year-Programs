My First Year Python Project.<br>
This repository contains my Python assignments and experiments.<br>

How to run(Windows):
1. Create the environment: `python -m venv (VENV Name)`
2. Activate the environment: `.\(VENV Name)\Scripts\activate.bat`
3. Install requirements: `pip install -r STDREQ.txt`
4. Run the desired programs: python "Desired Program Name"

How to run(MacOS):
1. Create the environment: `python3 -m venv (VENV Name)`
2. Activate the environment: `source (VENV Name)/bin/activate`
3. Install requirements: `pip install -r STDREQ.txt`
4. Run the desired programs: python3 "Desired Program Name"
